Upload Existing Project ไปบน Github
===================================
1. สร้าง New Repository บน Github
2. ใส่คำสั่ง git init -b master
3. เพิ่ม File .gitignore
4. ใส่คำสั่ง git add .
5. ใส่คำสั่ง git commit -m "First Commit"
6. ใส่คำสั่ง git remote add origin https://github.com/Sanya-Putthikarnchanakul/NodeTutorial.git
7. ใส่คำสั่ง git push origin master
